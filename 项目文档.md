### 桥架计算器 Web 版（开源/免费版）项目设计文档

#### 1. 产品愿景

打造一个**移动端优先、纯前端、无广告、可离线使用**的桥架弯头计算工具。打破行业信息差，让所有电工师傅都能免费用到“高级计算”功能。

#### 2. 核心技术栈建议

为了保证零成本运营和极致的加载速度，建议采用**纯前端架构**（不需要后端服务器和数据库）：

- **前端框架**：Vue 3 (配合 Composition API) 或 React。如果想极简，甚至原生 HTML + JS 也能搞定。
- **UI 组件库**：Vant (Vue的移动端组件库) 或 Tailwind CSS（方便快速搭建响应式界面，尤其适合手机端）。
- **图形绘制 (进阶)**：HTML5 Canvas 或 SVG（用于动态生成带有尺寸标注的桥架切口图）。
- **部署与托管**：GitHub Pages、Vercel 或 Netlify（这三者对静态网站都是永久免费的）。
- **核心特性**：PWA (Progressive Web App) 技术。加上 PWA 后，用户可以在浏览器里点击“添加到主屏幕”，网站就会像原生 App 一样存在手机桌面上，且**支持无网离线使用**（这对于地下室施工极其重要）。

#### 3. 功能模块划分 (MVP 最小可行性产品)

**模块一：基础计算 (引流与日常高频)**

- **万能爬坡计算**：输入角度、爬坡高度。输出：斜边长、切口尺寸。
- **水平弯计算**：输入角度、平移宽度。输出：斜边长、切口尺寸。

**模块二：特殊计算 (核心竞争力/替代VIP)**

- **多层过桥**：输入角度、层间距。输出：每层递增的斜边补偿值。
- **大小头变径**：输入变化宽度、过渡长度。输出：对角线切割长度。
- **死角弯避让**：输入障碍物尺寸。输出：提前起弯的距离计算。

**模块三：图纸与可视化 (提升用户体验)**

- **动态切口图**：根据计算结果，动态渲染出一个带有红线标注的 V 型切口示意图，直观告诉用户“在哪里画线”、“锯掉哪里”。

#### 4. 核心算法库 (Math / Core Logic)

在代码里，你需要把所有的数学计算封装成一个纯函数库 `trayMath.js`。

*注意：JavaScript 中的 `Math.sin()` 等函数接收的是弧度（Radians）而不是角度（Degrees），需要先做转换。*

- **角度转弧度**：$R = \text{angle} \times \frac{\pi}{180}$
- **斜边计算公式**：$\text{Hypotenuse} = \frac{\text{Height}}{\sin(R)}$
- **切口宽度公式**：$\text{CutWidth} = 2 \times \text{EdgeHeight} \times \tan(\frac{R}{2})$

#### 5. 页面与交互设计规范

- **大字体、高对比度**：工地环境通常光线复杂（太亮或太暗），师傅们可能还戴着手套或护目镜。输入框要大，结果展示必须极其醒目（建议用加粗大号字 + 高亮背景色）。
- **一键复制**：计算出结果后，提供一个“复制数据”按钮，方便师傅们发微信或记录在备忘录里。
- **输入联动**：角度选择最好做成 30°、45°、60°、90° 的快速按钮（Quick Buttons），而不是纯键盘输入，减少操作步骤。
